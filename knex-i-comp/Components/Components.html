KNEX-I-COMP: Components and Subassemblies

Here follow brief descriptions of all of the components and subasssemblies of the knex-i-comp model.

Bit Frame
The bit frame is a major structural component of the knex-i-comp. There is one bit frame for each control stage and each state bit.
Actuator Rod
analogous to the Digi-Comp I's logic rods.
Sense Rod
analogous to the Digi-Comp I's clock rods.
Bit Assembly
analogous to the Digi-Comp I's flip-flops.
Programming Hardware
Attatched to the bit assembly, the thing that the yellow struts that the actuator rods engage with mount on. each bit assembly, there is one of these for each actuator rod. The thing that the sense rods engage with, composed of two red connectors and a green strut. There are two of these on each bit assembly for each sense rod.
Control/Actuator Linkage
Sense Rod Weight
Actuator Rod Counterweight
There is one actuator rod counterweight for each actuator rod assembly. The counterweight serves to pull the actuator rod back to its normal position and hold it there.
Bit State Indicator
One mounted on the end of each bit assembly. Obstructed by the Bit State Mask when the bit is in the zero state, partially visible when the bit is in the one state.
Bit State Mask
one per bit assembly. Mounted on the bit frame.
Sense Rod Lifter
Sense Rod Lifter Linkage
So that the user can lower the sense rods (by lowering the sense rod lifter) from the front of the machine.
